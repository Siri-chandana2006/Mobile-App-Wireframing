
# Design Thinking Process

## Empathize
Interviewed students to understand their challenges in sharing and accessing notes.

## Define
Students need a centralized platform to access academic materials.

## Ideate
Features considered:
- Upload Notes
- Download Notes
- Search Functionality
- User Profiles

## Prototype
Created low-fidelity wireframes using Figma.

## Test
Collected feedback from classmates and improved navigation.

## Conclusion
The proposed application simplifies note sharing and access to study materials.
